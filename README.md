## 🚀 Using the Application

The interface splits into three main segments. Each segment controls a part of your artificial intelligence workflow.

### Managing Models
The model manager allows you to import files from your computer. Click the browse button to select a model file. The application displays the status of each loaded model. You can remove models that you no longer need to free up storage space.

### Configuring Runtimes
The runtime settings adjust how the computer processes data. You can decide how much of your computer power the software uses. If your computer slows down during use, lower the thread count in this menu. These settings ensure the software runs on various hardware builds.

### Coding Workflows
The workflow tab lets you send prompts and receive text outputs. You type your query in the top box. The application processes the request. The text appears in the bottom window. You can save your work as a text file for later use.

## 🔧 Frequently Asked Questions

### Why does the software run slowly?
Large models require high-performance hardware. If you notice delays, try using a smaller model file. Ensure you have the latest drivers for your graphics card installed.

### Where do I find model files?
Users typically download models from open research websites. Ensure you download files in the correct format for this application. Place your downloaded files in a folder you can easily locate.

### How do I update the software?
The application checks for updates when you start it. If an update exists, click the update button. The software performs the update automatically. You do not need to uninstall the current version first.

### Does the software require an internet connection?
Once you install the software and download your models, you can run everything offline. You only need the internet to download the application or new models. This ensures total privacy for your data.
